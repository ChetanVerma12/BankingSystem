# Banking System

This project is a simple banking system implemented in Java using JDBC and MySQL. It allows users to register, log in, and open bank accounts.

## Features

- User registration and login
- Open a new bank account with an initial balance
- Check if a user or account already exists
- Secure account operations with security pins

## Project Structure

- **User Management**: Handles user registration and login.
- **Account Management**: Manages bank account creation and checks for account existence.
  
## Technologies Used

- Java
- JDBC (Java Database Connectivity)
- MySQL
